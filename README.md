## Prepare
You need to have a [Telegram bot](https://core.telegram.org/bots/api).
After this, set `.env` file with the following variables:

```BOT_TOKEN=
BOT_NAME=
TONWEB_URL=
TONWEB_API=```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
